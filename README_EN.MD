Application flow:

1. User sends a message through the web chat.
2. Next.js calls `/api/chat`.
3. The route forwards the message to the Python backend.
4. The backend chooses between:
   - Calculator  
   - Currency  
   - Crypto  
   - Weather  
   - Or the LLM  
5. The final answer is displayed in the chat UI.
